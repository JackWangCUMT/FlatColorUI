FlatColorUI

FlatColorUI is a Library of Flat Colors for Unity3D.

Setup:

  1. Create an Editor Folder inside your project
  2. Place FlatColorUI.cs in the newly created Editor Folder
  3. Window -> FlatColorUI to open the editor window
